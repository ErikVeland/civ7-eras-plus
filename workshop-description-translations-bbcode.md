# Eras+ Workshop Description Translations

## Simplified Chinese

[h1]Eras+[/h1]

Eras+ 在当前模组 API 允许的范围内，为 Civilization VII 添加时代节奏预设。

可选择原版节奏、更强的延长曲线、较轻的 Epic 研究预设、适合多人游戏的曲线，或仅延长现代时代。

[h2]Pacing presets[/h2]
[list]
[*][b]Vanilla：[/b] Eras+ 不改变时代节奏。
[*][b]Balanced Extended：[/b] 更强的单人延长节奏。时代更长，科技/市政更慢，轻微抑制单位海，并加快道路。
[*][b]Balanced Extended+：[/b] 推荐默认值。科技和市政成本接近 Epic 风格，同时生产、成长、维护费和胜利项目成本接近标准。
[*][b]MP Pace：[/b] 为多人游戏使用更短阈值，避免玩家进入可重复科技/市政后时代仍拖得太久。
[*][b]Modern Stretch：[/b] 不改变古典和探索时代，然后大幅提高现代时代阈值和现代科技/市政成本。
[*][b]自定义（分时代）:[/b] 独立设置每个时代（远古、探索、现代）的长度。
[/list]

[h2]What presets can change[/h2]
[list]
[*]时代点数阈值。
[*]科技和市政成本。
[*]里程碑和可重复未来科技/市政带来的时代进度点数。
[*]在所选预设需要时调整成长、单位海压力、道路、铁路和现代胜利项目成本。
[/list]

这是会影响存档的玩法模组。请在游戏设置中选择预设，并尽量开始新游戏。

灵感来自早期 Civilization 游戏的 Historic Speed / Extended Eras 节奏。

完全兼容 ToT 1.4.0+。
