# Eras+ Workshop Description Translations

## English

Eras+ extends Civilization VII Ages within the limits of the current modding API.

It keeps production close to Standard while stretching the parts of the game that make Ages race by: tech and civic costs, Age point caps, milestone Age pressure, future tech/civic pressure, growth pacing, unit spam pressure, and road movement.

[h2]Current tuning[/h2]
[list]
[*]Much higher Age point caps: 240 / 300 / 360 for abbreviated, standard, and long Ages.
[*]Slower techs and civics, scaling upward from Antiquity to Modern.
[*]Reduced Age pressure from milestones and repeatable future tech/civic completions.
[*]Slightly slower settlement growth.
[*]+10% production cost and +1 maintenance for trainable combat units.
[*]Faster roads and railroads so longer Ages feel more mobile, not more static.
[*]Modern victory projects are modestly slower without blocking victory routes.
[/list]

This is a save-affecting gameplay mod. Start a new game for best results.

Inspired by Historic Speed / Extended Eras style pacing for earlier Civilization games.

Fully compatible with ToT 1.4.0+.
